# BPMN-схемы для туроператора

## 1. Бронирование тура
Процесс: Поиск тура → Проверка доступности → Заполнение заявки → Оплата → Подтверждение → Выписка документов

## 2. Бронирование отеля  
Процесс: Поиск отеля → Проверка номеров → Блокировка номера → Оформление брони → Подтверждение

## 3. Покупка билетов
Процесс: Поиск рейсов → Выбор рейса → Ввод данных → Оплата → Выписка билета

## 4. Поддержка клиентов
Процесс: Получение обращения → Фиксация в CRM → Определение типа → Обработка → Ответ → Закрытие
